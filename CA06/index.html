Studio Shooting



This project involves the creating an advertisement for new internet radio station. The advertisement will be used for various public transport platforms as well as on the internet platform Spotify.

For this task, I choose a radio station under the Music: Independent "up and coming" artists operating in the genre indie electronica.

I shoot a photo of myself as a model for the advertisement using an adhoc home studio set up that consist of Canon DSL camera and tripod.

I choosed a name AREWA BANGA for the adverstisement because it sound African and is relatable to new groups of radio listeners, I wanted to attract with the music advertisement. It also matches the personality of the model used.

I used the font “Adhesive Nr. Seven” and a font size of 130pt for the AREWA BANGA on the design. Also, I used the “Microsoft sans serif” and font size of 48pt for the showing on radio inscription. For the Indie electronica inscription I used font named “Angie open pro” with font size of 60pt. The typography fits well with the design and gives the design a modern look and feel.
